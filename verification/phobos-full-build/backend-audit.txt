Build ordering

1. Jabara and Orgasm bootstrap.
2. Bootstrap Juicer/Pad, Haruka, and Phobos.
3. Bootstrap Haruka + bootstrap Phobos self-build Phobos.
4. Phobos rebuilds Haruka.
5. Final Haruka + Phobos rebuild Phobos for host and FruityOS.
6. Phobos compiles Mars from sol/mars for host and FruityOS.
7. Phobos compiles Peel, Yuzu, Pulp, and other Haruka-language components.

Mars integrity

sol/mars has no source diff from repository HEAD.
Mars and Phobos are compiled from separate source directories.
No build or rename script copies a Phobos output to a Mars output or vice versa.
